# Погодное приложение

Простое веб-приложение для просмотра погоды на сегодня и на неделю.

## 🚀 Возможности

- Погода на сегодня (температура, описание, влажность, ветер)
- Прогноз на 7 дней
- Выбор города из списка

## 🏙️ Доступные города

- Москва
- Санкт-Петербург
- Петрозаводск
- Нижний Новгород
- Казань
- Екатеринбург
